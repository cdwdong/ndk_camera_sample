# ndk_camera_sample
 
회사 현장실습 개인 프로젝트

ndk를 사용하여 카메라 프리뷰를 opencv 사진변환,  
ImageReader를 Camera2 API에 등록시켜서 받아온 이미지 SurfaceView에 Opengl을 이용하여 렌더링

문제가 될까봐 샘플만 올렸습니다.
